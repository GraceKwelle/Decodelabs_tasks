# Project 2: Exploratory Data Analysis (EDA)
**DecodeLabs Data Analytics Internship**

## Overview
Analyzed 1,200 e-commerce orders to uncover pricing trends, customer 
behavior patterns, and high-value outliers.

## Key Findings
- Right-skewed TotalPrice distribution (Mean $1,053.97 > Median $823.62)
- 8 statistical outliers above $3,330.41, led by ORD200789 ($3,456.40)
- Chairs are the top revenue category; Phones the lowest
- Instagram leads referral traffic; Online Banking leads payment methods

## Method
Descriptive statistics and IQR-based outlier detection in Excel, 
visualized with box-and-whisker and bar charts, summarized in an 
Executive Summary (Problem, Methodology, Findings, Recommendations).

Microsoft Excel
